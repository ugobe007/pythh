## What Pythh is

Pythh is an **intent detection platform**, not a database.

Most data tools (Crunchbase, PitchBook, ZoomInfo) track *static facts* — funding rounds, headcount, valuation. Pythh tracks **language → intent → action**:

- What companies **say** in the wild (news, hiring, press, product launches)
- What those words **signal** about their next strategic move
- Who should **care** about that signal, and **why now**

That language often appears **6–18 months before** major events: fundraises, acquisitions, enterprise deals, and distress situations.

For Pythiam, the output is **pre-scored deal flow matched to our thesis** — ranked by signal quality and timing — not a raw list of inbound decks.

**Pipeline flow:**

```
discovery → entity gate → GOD score → signal dimensions → trajectory → match → monitor
```

## The platform stack

Six production layers on [pythh.ai](https://pythh.ai):

| Layer | Function |
|-------|----------|
| **Entity resolution gate** | Name validation, URL checks, junk filtering — noise never becomes deal flow. |
| **GOD scoring (0–100)** | Seven-pillar composite (team, traction, market, product, vision, grit, momentum) for comparable selection. |
| **Signal intelligence** | News, hiring, funding cues, product velocity parsed into classified events. |
| **Trajectory engine** | Ordered sequences (hire → GTM → diligence → raise) predict what happens next. |
| **Thesis matching** | Stage, sector, check size, timing — ranked mandate fit, not spray-and-pray lists. |
| **Portfolio monitoring** | Post-investment signal refresh, event logging, health tiers — ongoing diligence. |
